## Current Focus

### Valer!a

Valer!a is an alpha multimodal assistant designed to coordinate:

- conversation;
- one-shot scene inspection;
- local speech input and output;
- assisted physical-task workflows;
- runtime state, telemetry, and recovery guidance.

The project explores how an assistant can help with physical tasks while
keeping the real-time loop local, observable, and explicit about uncertainty.

## Engineering Interests

- Multimodal AI systems
- Computer vision workflows
- Human-in-the-loop assistants
- Local-first prototypes
- Tool orchestration and runtime state
- Practical product engineering

## Principles

- Build useful systems before polishing abstractions.
- Keep private credentials and local environment details out of public repos.
- Treat AI output as assistance, not unchecked truth.
- Make uncertainty visible when a system cannot verify something safely.
